File operations

ruby: require 'fileutils'

Creating an empty file

bash: touch File.txt
ruby: FileUtils.touch 'File.txt'

Creating an file

bash: echo "File contents" > File.txt
ruby: File.open('File.txt', 'w') {|f| f.write("File contents") }

Creating a directory

bash: mkdir -p path/to/new
ruby: FileUtils.mkdir_p 'path/to/new'

Deleting a file

bash: rm 'File.txt'
Java: new File("File.txt").delete();
